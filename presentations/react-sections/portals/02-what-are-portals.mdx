## What are Portals?

### Portals allow you to:
- **Render children** into a DOM node outside the parent
- **Break out** of CSS overflow/z-index constraints
- **Maintain** React event bubbling
- **Create** modals, tooltips, and overlays
- **Preserve** component context and state

### Common Use Cases:
- Modal dialogs
- Tooltips and popovers
- Notifications/toasts
- Dropdown menus
- Full-screen overlays
