## Summary

### Key Concepts:
- **createPortal** renders outside parent DOM
- **Event bubbling** follows React tree
- **Focus management** for accessibility
- **Position calculation** for tooltips/dropdowns
- **Cleanup** on unmount

### When to Use Portals:
- Modals and dialogs
- Tooltips and popovers
- Notification systems
- Dropdown menus
- Breaking out of overflow constraints

### Remember:
- Portals maintain **React context**
- Events **bubble through React tree**
- Always consider **accessibility**
- Manage **z-index** carefully
