## What is Conditional Rendering?

### Render different UI based on:
- **Habit completion status** - Show checkmark or empty circle
- **Streak milestones** - Display celebrations for achievements
- **Time of day** - Show relevant habits for morning/evening
- **User preferences** - Hide/show completed habits
- **Loading states** - Display skeleton while fetching habits
- **Empty states** - Show onboarding when no habits exist

### Techniques:
- If/else statements
- Ternary operators
- Logical && operator
- Switch statements
- Component composition
