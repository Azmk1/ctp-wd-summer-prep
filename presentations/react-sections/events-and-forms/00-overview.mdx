## Event Handling in React

### Key Concepts:
- **Synthetic Events** - Cross-browser wrapper around native events
- **Event Handlers** - Functions that respond to user interactions
- **Event Object** - Contains event information
- **Event Delegation** - React handles events at root level
- **Automatic Binding** - Arrow functions bind `this` automatically

### Common Events:
- Click, Double Click, Mouse Events
- Keyboard Events
- Form Events
- Focus Events
- Touch Events
