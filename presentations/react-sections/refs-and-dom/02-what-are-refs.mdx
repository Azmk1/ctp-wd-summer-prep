## What are Refs?

### Refs provide:
- **Direct access** to DOM elements
- **Persistent values** that don't trigger re-renders
- **Imperative API access** to child components
- **Integration** with third-party DOM libraries
- **Performance** optimizations

### Common Use Cases:
- Focus, text selection, media playback
- Animations and measurements
- Integrating non-React libraries
- Storing mutable values
- Previous value tracking
