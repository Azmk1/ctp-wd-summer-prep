## Compound Components

Compound Components work together to form a complete UI component. They share implicit state and provide a clean API where child components are used as building blocks. Examples include tabs, accordions, and menus where multiple parts need to coordinate.

</SlideLayout.Full>
<SlideLayout.Full>

### Key Concepts:
- **Multiple components** work together as one unit
- **Implicit state sharing** via Context
- **Flexible composition** with clean JSX
- **Examples**: Tabs, Accordions, Select menus

</SlideLayout.Full>
