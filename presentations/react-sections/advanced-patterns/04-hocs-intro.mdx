## Higher-Order Components (HOCs)

A Higher-Order Component is a function that takes a component and returns a new component with additional functionality. HOCs are useful for adding cross-cutting concerns like authentication, loading states, or data fetching to multiple components.
</SlideLayout.Full>
<SlideLayout.Full>
### Key Concepts:
- **Function that takes a component** and returns enhanced component
- **Adds behavior** without modifying original component
- **Composable** - can chain multiple HOCs
- **Common uses**: authentication, logging, data fetching
