## What are Server Components?

### Server Components enable:
- **Zero bundle size** for server-only code
- **Direct database access** from components
- **Automatic code splitting** per component
- **Progressive rendering** with streaming
- **Better performance** with less JavaScript

### Key Differences:
- **Server Components**: Run only on server, no JS sent to client
- **Client Components**: Traditional React components
- **Shared Components**: Can run on both server and client
