## Generic Syntax and Patterns

**Generic Classes:**
```typescript
class Container<T> {
  private items: T[] = [];
  
  add(item: T): void {
    this.items.push(item);
  }
  
  get(index: number): T | undefined {
    return this.items[index];
  }
}
```

**Generic Functions:**
```typescript
function identity<T>(value: T): T {
  return value;
}

function getFirst<T>(items: T[]): T | undefined {
  return items[0];
}
```
