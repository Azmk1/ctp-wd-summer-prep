## Common Closure Pitfalls

**Watch out for these gotchas:**

1. **Loop Variable Capture**
```javascript
// ‚ùå Problem: All buttons alert "3"
for (var i = 0; i < 3; i++) {
  button[i].onclick = function() {
    alert(i); // Always 3!
  };
}
```
