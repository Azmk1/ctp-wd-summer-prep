## Example Flow: Creating a Habit

1. User submits habit form (Frontend)
2. Frontend sends POST request to `/api/habits` (Backend)
3. Backend validates data and saves to database
4. Database returns saved habit with ID
5. Backend sends success response to Frontend
6. Frontend updates UI to show new habit
</SlideLayout.Full>
